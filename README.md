# VISE_Scripts
PowerShell scripts for my Video Integrated Scripting Environment (VISE) project supporting live event and streaming broadcast automation

Main.ps1 is an example of automating Blackmagic Designs ATEM vision mixer with X-keys keyboard. It includes a monitor function to update switch LED’s if the switcher state changes from another control method.

See https://ianmorrish.wordpress.com for more information
